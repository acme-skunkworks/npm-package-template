# Acme Skunkworks NPM package template

A GitHub Template repository for Acme Skunkworks npm packages — a minimal, buildable
pnpm + TypeScript ESM skeleton plus the shared CI / release shell, so a new package can be
generated and released without rebuilding the infrastructure each time. Click **Use this
template** to spawn a new package, then follow the generation checklist in
[`CLAUDE.md`](CLAUDE.md).

## Setting it up

The one-time org/repo settings that turn this into a working GitHub Template repository —
and the settings every spawned repo inherits — live in
[`docs/TEMPLATE-BOOTSTRAP.md`](docs/TEMPLATE-BOOTSTRAP.md).
